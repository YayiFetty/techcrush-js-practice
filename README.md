

**Q1::** Declare three variables using `const` and `let`: your name (string), your age (number), and a boolean called `isStudent`. Log all three to the console on separate lines.
**Question was testing knowledge based on:** Variables & Data Types — understanding `const` vs `let` and basic data types (string, number, boolean).


**Q2::** Write an arrow function called `multiply` that takes two numbers and returns their product. Call it with 6 and 7 and log the result.
**Question was testing knowledge based on:** Arrow Functions — writing concise functions and returning values.


**Q3::** Write a function called `greetUser` that takes a name parameter and returns `'Welcome to TechCrush, [name]!'`. Test it with your own name.
**Question was testing knowledge based on:** Functions & Strings — passing parameters and using template literals.


**Q4::** Given an array `["mango", "orange", "banana"]`, add `'pineapple'` to the end, remove the first item, and log the updated array and its length.
**Question was testing knowledge based on:** Arrays — using `push()` and `shift()` to modify arrays.


**Q5::** Use `map()` on `[3, 6, 9, 12, 15]` to create a new array where every number is doubled. Log the new array without changing the original.
**Question was testing knowledge based on:** Array Methods — transforming arrays with `map()` without mutation.



**Q6::** Create an object `myProfile` with name, age, country, and hobby. Write a function `displayProfile(person)` that logs a sentence using those properties.
**Question was testing knowledge based on:** Objects — creating objects and accessing properties with dot notation.



**Q7::** Write a function `formatName(firstName, lastName)` that trims spaces and returns `'LASTNAME, Firstname'`.
**Question was testing knowledge based on:** String Methods — using `trim()`, `toUpperCase()`, and string formatting.



**Q8::** Write a function `gradeScore(score)` that returns grades (70–100 A, 60–69 B, 50–59 C, 40–49 D, below 40 F). Test with sample scores.
**Question was testing knowledge based on:** Control Flow — using `if / else if / else` correctly.


**Q9::** Write a `for` loop to print the multiplication table of 5 from 1 to 10 in the format `5 x 1 = 5`.
**Question was testing knowledge based on:** Loops — using `for` loops and template literals.


**Q10::** Write a function `summarizeStudent` that takes an array of students and returns strings like `'Amara scored 88 — Grade: B'` using `gradeScore`.
**Question was testing knowledge based on:** Combining Concepts — arrays, objects, functions, and reusability.
